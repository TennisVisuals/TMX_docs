## Administrator Configuration

Administrators for some organizations are granted access to data storage configuration.

{% include image.html file="ch_tmx_datastorage.png" alt="Data Storage" caption="Data Storage" %}

For smaller organizations who don't have an existing player repository to serve as a "master record", data storage facilitates the use of [Google Sheets](tmx_google_sheets.html) as a repository which all organization referees and administrators may synchronize with their local databases.

{% include image.html file="ch_tmx_sheet_data.png" alt="Data Storage" caption="Data Storage" %}
